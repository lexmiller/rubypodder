rubypodder

Overview

Rubypodder is a simple podcast aggregator inspired by bashpodder. You configure it with all the podcasts (radio programmes, audio blogs, etc.) that you regularly listen to. Then you run rubypodder as a cron job and each morning before setting off, copy anything new onto your mp3 player.

Download and Installation

Note: ruby and rubygems must be installed on your computer. On my Mandriva desktop, this would be:
# urpmi ruby rubygems rdoc
Download and install rubypodder:
# gem install rubypodder

Running rubypodder

Run rubypodder from the command line like this:
$ rubypodder
Rubypodder comes with a default configuration file containing an example podcast (the BBC comedy programme "The Now Show"). If there were any episodes of this available you should see them in the directory:
~/.rubypodder/2007-01-27/
or whatever the date is when you run it. If there are no recent episodes, you won't see the date directory but you should see:
~/.rubypodder/rp.conf
~/.rubypodder/rp.log
~/.rubypodder/rp.done

Configuring rubypodder

You can edit rp.conf to put in the podcast feeds to which you wish to subscribe. My configuration file looks like this:
$ cat ~/.rubypodder/rp.conf
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio1/radio1smusicinterviews/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio4/inourtime/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/worldservice/documentaryarchive/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/sixmusic/talking6music/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio4/thenowshow/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio4/broadcastinghouse/rss.xml
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio4/radio4choice/rss.xml
http://feeds.feedburner.com/rubyonrailspodcast
http://downloads.bbc.co.uk/rmhttp/downloadtrial/radio1/radio1sentertainmentnews/rss.xml
http://www.guardian.co.uk/podcasts/comedy/rickygervais/mp3.xml
http://www.ibm.com/developerworks/podcast/channel-dwall.rss
$
You can set up rubypodder to run as a cron job. I set it up to run at 5am each morning. On my computer to do this you edit your crontab with:
crontab -e
and add the line
0 5 * * * rubypodder
